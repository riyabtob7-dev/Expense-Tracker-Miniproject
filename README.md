# 💰 Expense Tracker (Python GUI)

A simple and user-friendly Expense Tracker application built using **Python** and **Tkinter**. This application allows users to add, view, and calculate their expenses through a graphical user interface (GUI). All expense data is stored in a text file so it remains available even after closing the application.

---

## 📌 Features

- ✅ Add new expenses
- ✅ View all saved expenses
- ✅ Calculate total spending
- ✅ User-friendly GUI using Tkinter
- ✅ Stores data in a text file
- ✅ Loads previously saved expenses automatically
- ✅ Input validation for amount field

## ▶️ How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open the project folder in VS Code.
4. Run the following command:

```bash
python expense_tracker.py
```

5. Start adding and managing your expenses.
